# 🏥 Go Ecosystem DH

**Soluciones tecnológicas para el ecosistema de salud digital**

---

## 🌐 Acerca de Nosotros

Go Ecosystem DH es una organización dedicada al desarrollo de soluciones tecnológicas innovadoras para el sector salud en **Colombia**. Nuestro enfoque está en sistemas hospitalarios, registro clínico, gestión administrativa y herramientas digitales que mejoran la atención al paciente.

## 🛠️ Stack Tecnológico

| Área | Tecnologías |
|------|------------|
| **Backend** | Go, Node.js, .NET |
| **Frontend** | Angular, React |
| **Móvil** | Android (Kotlin/Java) |
| **Infraestructura** | Docker, Docker Compose, GitHub Actions |
| **Base de Datos** | SQL Server, PostgreSQL |
| **CI/CD** | GitHub Actions, Self-hosted Runners |

## 📦 Proyectos Principales

| Proyecto | Descripción |
|----------|-------------|
| **HIS (Hospital Information System)** | Sistema integral de información hospitalaria — servicios, gestión clínica y cirugía |
| **RCF (Registro Clínico Fácil)** | Aplicación de registro clínico con soporte web, API y móvil |
| **Costos y CMI** | Cuadro de mando integral y gestión de costos hospitalarios |
| **Cero Trámites** | Plataforma de eliminación de trámites para pacientes |
| **Mipres** | Integración con el sistema MIPRES del Ministerio de Salud |
